About Our Project

Detecting Fake News with Python and Machine Learning

Type of Project ?

This is a binary classification problem which can be solved using a classifier that detects whether the news is fake or not.

What type of Algorithm ?

We apply the Multinomial Naive Bayes algorithm to the preprocessed text and train and evaluate our model on the dataset

Information about the dataset?

The dataset for this project contains two directories inside it. One contains the true (correct) news and the other contains the fake news. Summing both of them up, the total dataset comprises 44,898 instances.
